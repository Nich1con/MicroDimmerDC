MicroDimmerDC
============
Простой и очень компактный DC диммер для светодиодных лент и панелей с напряжением 12В
Проект на EasyEda: https://oshwlab.com/Nich1con/nanopwmmodule

### Возможности
* **Управление яркостью, включение и выключение при помощи сенсора**
* **Ток нагрузки до 5А (зависит от силового ключа)**
* **Частота ШИМ 30кГц (Не стробит на камеру)**
* **CRT гамма по квадратной параболе**

### Список компонентов 
* x1 Микроконтроллер ATtiny10 (SOT-23-6)
* x1 Контроллер сенсора TTP223E (SOT-23-6)
* x1 LDO стабилизатор 78L05
* x1 N-FET IRLML0030TRPBF (SOT-23)
* x1 L0805 LED
* x1 R0805 1k
* x2 R0805 10k
* x1 R0805 220R
* x2 C0805 1uF
* x1 C0805 22pF
